# downloadLUH2v2h

Download the LUH2v2h dataset (states.nc, transitions.nc, management.nc,
staticData_quarterdeg.nc).

## Usage

``` r
downloadLUH2v2h(subtype = NULL)
```

## Arguments

- subtype:

  one of states, management, transitions, cellArea

## Value

metadata list with URL, DOI, title, description, author, unit, version,
release date

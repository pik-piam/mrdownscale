# readMagpieFulldataGdx

Read function for data coming from the MAgPIE model.

## Usage

``` r
readMagpieFulldataGdx(subtype)
```

## Arguments

- subtype:

  type of data to be read in. Available options are land, crop,
  woodHarvestWeight, woodHarvestArea, fertilizerRegional, clustermap
